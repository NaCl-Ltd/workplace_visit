# NaCl

* 英文社名: Network Applied Communication Laboratory Ltd.
* 通称: NaCl
  * 塩化ナトリウムの化学式
  * ほとんどの生物にとって生命維持に不可欠なナトリウム源
  * 「IT分野において必要不可欠な存在でありたい」

# 業務内容
